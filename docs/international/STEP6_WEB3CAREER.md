## 💼 面试准备（如果收到回复）

### 技术问题准备
```
常见问题：
1. "Explain how a DEX works"
   → 答：AMM formula (x*y=k), liquidity pools, price impact

2. "What are reentrancy attacks?"
   → 答：Explain the attack + how to prevent (ReentrancyGuard)

3. "Difference between Web3.js and Ethers.js?"
   → 答：Both are libraries to interact with Ethereum, different API design

4. "Explain gas optimization techniques"
   → 答：Use events, pack structs, minimize storage operations, etc.
```

### 项目介绍准备
```
准备讲3个项目（每个5分钟）：
1. DeFi Trading Platform
2. NFT Marketplace
3. Web3 Analytics Dashboard

每个项目介绍结构：
- Problem: What problem did you solve?
- Solution: What did you build?
- Tech Stack: What technologies?
- Result: What was the outcome? (metrics)
- Challenges: What was hard?
```

### 薪资谈判准备
```
你的目标价位：
- Contract: $120-150/hour
- Full-time: $150-200K/year

谈判技巧：
- 先听对方出价
- 基于市场价+你的经验
- 不要 afraid to negotiate
- 可以问："Is this flexible?"
```
